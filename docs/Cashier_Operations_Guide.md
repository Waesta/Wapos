# Daily Workflow

## üåÖ Opening Procedures

### 1. System Login
- Enter your username and password
- Verify you're logged into the correct location/terminal

### 2. Cash Drawer Setup
- Count your opening float
- Verify it matches the expected amount

## üíº During Service

### Opening Tabs

**For Walk-in Guests:**
1. Click **[+ New Tab]**
2. Select **"Name"** tab type
3. Enter guest's name
4. Select your bar station
5. Select the waiter (if using waiter assignment)
6. Click **[Open Tab]**

**For Card Pre-Authorization:**
1. Click **[+ New Tab]**
2. Select **"Card"** tab type
3. Enter last 4 digits of card
4. Enter pre-auth amount
5. Click **[Open Tab]**

**For Hotel Guests:**
1. Click **[+ New Tab]**
2. Select **"Room"** tab type
3. Select the room from dropdown
4. Click **[Open Tab]**

---

### Adding Items

1. **Select the tab** from the left panel (it highlights when selected)
2. **Find the product:**
   - Use category tabs at top
   - Or type in search box
3. **Click the product** to add it

**For Portioned Items (Spirits, Wine):**
- A popup appears with portion options
- Select: Tot (25ml/35ml/50ml), Shot, or Glass
- Adjust quantity if ordering multiple
- Click **[Add to Tab]**

**For Regular Items:**
- Click product ‚Üí enters quantity popup
- Adjust quantity if needed
- Click **[Add]**

---

### Modifying Orders

**Remove Item:**
1. Click the item in the order list
2. Click **[Void Item]**
3. Enter reason
4. Confirm

**Change Quantity:**
1. Click the item
2. Adjust quantity
3. Save changes

**Add Special Instructions:**
1. Click the item
2. Enter notes (e.g., "no ice", "extra lime")
3. Save

---

### Applying Discounts

1. Select the tab
2. Click **[Discount]** button
3. Enter discount:
   - **Amount**: Fixed amount off (e.g., 500)
   - **Percentage**: Percent off (e.g., 10%)
4. Select reason from dropdown
5. Click **[Apply]**

*Note: Some discounts may require manager approval*

---

### Adding Tips

1. Select the tab
2. Click **[Add Tip]** button
3. Enter tip amount
4. Click **[Save]**

---

## üí≥ Processing Payments

### Cash Payment

1. Select tab ‚Üí Click **[Pay]**
2. Select **"Cash"**
3. Enter **amount received** from customer
4. System shows **change due**
5. Click **[Complete Payment]**
6. Give change to customer
7. Print receipt if requested

### Card Payment

1. Select tab ‚Üí Click **[Pay]**
2. Select **"Card"**
3. Process card on terminal
4. Wait for approval
5. Click **[Complete Payment]**
6. Print receipt

### M-Pesa Payment

1. Select tab ‚Üí Click **[Pay]**
2. Select **"M-Pesa"**
3. Enter customer's phone number (format: 07XXXXXXXX)
4. Click **[Send Request]**
5. Customer receives prompt on their phone
6. Customer enters PIN to confirm
7. Wait for confirmation
8. Click **[Complete Payment]**

### Room Charge

1. Select tab ‚Üí Click **[Pay]**
2. Select **"Room Charge"**
3. Select room from dropdown
4. Verify guest name matches
5. Click **[Complete Payment]**
6. Charge posts to room folio

### Split Payment

1. Select tab ‚Üí Click **[Pay]**
2. Click **"Split Payment"**
3. Enter first payment amount and method
4. Process first payment
5. Enter second payment amount and method
6. Process second payment
7. Continue until total is covered

### Transferring a Tab

*When a waiter hands off to another waiter:*

1. Select the tab
2. Click **‚ãÆ** ‚Üí **"Transfer Tab"**
3. Select the new waiter
4. Enter reason (optional)
5. Click **[Transfer]**

### Voiding a Tab

*Only when absolutely necessary:*

1. Select the tab
2. Click **‚ãÆ** ‚Üí **"Void Tab"**
3. Enter reason (required)
4. Click **[Confirm Void]**

*Note: Voids are logged and may require manager approval*

### Filtering Tabs by Waiter

*If enabled in settings:*

1. Use the dropdown above the tabs list
2. Select a waiter's name
3. Only their tabs will show
4. Select "All Waiters" to see everything

## üåô Closing Procedures

### 1. Close All Open Tabs
- Review all open tabs
- Contact customers with unpaid tabs
- Process all remaining payments
- Void any abandoned tabs (with manager approval)

### 2. Run End of Day Report
1. Go to **Reports** ‚Üí **End of Day**
2. Review:
   - Total sales
   - Payment breakdown (cash, card, M-Pesa)
   - Voids and refunds
   - Tips collected

### 3. Cash Up
1. Count all cash in drawer
2. Subtract opening float
3. Compare to system total
4. Note any variance
5. Complete cash up in system

### 4. Secure Cash
- Place cash in safe or hand to manager
- Keep cash up slip for records

### 5. Log Out
- Click your name ‚Üí **Logout**
- Never leave system logged in unattended

## ‚ö†Ô∏è Troubleshooting

### System Shows "Offline"
1. Check Wi-Fi/internet connection
2. Refresh the page (F5)
3. If persists, contact IT

### Printer Not Working
1. Check printer is powered on
2. Check paper supply
3. Check cable connections
4. Try printing test page
5. Restart printer if needed

### Payment Declined
1. Ask customer to try again
2. Try different payment method
3. Check internet connection
4. For M-Pesa: verify phone number

### Can't Find a Product
1. Check spelling in search
2. Browse categories manually
3. Product may be out of stock
4. Contact manager to add product

### Tab Won't Close
1. Verify payment covers full amount
2. Check for pending items
3. Ensure no pending BOTs
4. Contact manager if stuck
